Ubuntu 安装 Hexo 记录

发表于 2016-07-26   |  

安装官网安装步骤安装，往往会出现问题。

  • ###安装git:
    $ sudo apt-get install git-core目前没出现过问题。

  • ###安装 Node.js
    官方推荐用nvm安装。Ubuntu下安装，看网友提示说要安装nodejs-legacy，安装方法：$ sudo apt-get install nodejs-legacy，详情https://github.com/hexojs/hexo/issues/580。

    ####安装nvm
    cURL:
    $ curl https://raw.github.com/creationix/nvm/master/install.sh | sh
    Wget:
    $ wget -qO- https://raw.github.com/creationix/nvm/master/install.sh | sh
    安装nvm时，执行这阳光命令分别都出现了问题，但多执行几次后安装成功了。
    其中遇到的一个问题就是输入nvm提示没有安装nvm，然后再次安装提示已经安装，疑似某个配置出现问题，没有找到该命令。查询资料得知，在~/.bashrc 文件里配置nvm，发现配置正确。
    export NVM_DIR="/root/.nvm" [ -s "$NVM_DIR/nvm.sh" ] && . "$NVM_DIR/nvm.sh" # This loads nvm
    最后输入source .bashrc奇迹般的好了。

  • ###安装hexo
    $ npm install hexo-cli -g，由于之前设置过系统代理，看错误log是不能连接网络，取消代理，依然不行，重启电脑好了。

  • 使用hexo

最后在浏览器输入localhost:4000就好了。
