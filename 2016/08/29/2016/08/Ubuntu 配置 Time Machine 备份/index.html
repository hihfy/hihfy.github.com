Ubuntu 配置 Time Machine 备份

发表于 2016-08-29   |  

原文：http://blog.e9china.net/share/yonglinuxdajiantimemachineeeubuntupianqincekeyong.html

安装

安装netatalk

sudo apt-get install netatalk

安装Avahi

Ubuntu 系统已经安装Avahi，这里就不用安装了

配置

配置netatalk

  • 更改备份目录

    sudo vim /etc/netatalk/AppleVolumes.default

将

  • 创建目录

    sudo mkdir /home/xxxx/TimeCapsule

  • 给予目录权限

    sudo chmod -R 777 /home/xxxx/TimeCapsule

配置Avahi

由于Ubuntu已经包含Avahi-deamon，这里直接进行配置，终端输入：

sudo vim /etc/avahi/services/afpd.service写入：

%h
_afpovertcp._tcp
548
_device-info._tcp
0
model=Xserve

保存退出就可以了。
