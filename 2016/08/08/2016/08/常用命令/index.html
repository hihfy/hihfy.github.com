常用命令

发表于 2016-08-08   |  

查看命令的路径

which ssh
/usr/bin/ssh 显示该命令

显示端口号是否启动

lsof -i:xx // xx是端口号

杀死程序

kill -9 pid // 进程pid

根据进程名字查看信息

ps -ef |grep ssh
