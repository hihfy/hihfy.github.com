Ubuntu phpmyadmin 安装与配置

|

安装php

sudo apt-get install php php-fpm

安装与配置Nginx

安装Nginx

$ apt-get install nginx

配置Nginx

修改/etc/nginx/sites-available/default文件里的/usr/share/nginx/html;改为：/var/www;、index后面添加index.php

sudo service nginx reload

hphmyadmin

安装hphmyadmin

// 安装hphmyadmin
sudo apt-get install phpmyadmin
// 创建软连接
sudo ln -s /usr/share/phpmyadmin/ /var/www/phpmyadmin
// 不使用时，删除软连接方式
sudo rm -rf /var/www/phpmyadmin

安装过程可能出现输入密码，这时会创建一个phpmyadmin的账户。
出现选择服务器是，可以选择Apache

配置phpmyadmin

修改/etc/nginx/sites-available/default文件内容，将root后面内容改为var/www,添加以下内容，不然会无法解析，会直接下载这个index.php这个文件。

location ~ \.php$ {
        try_files $uri =404;
        fastcgi_split_path_info ^(.+\.php)(/.+)$;
        fastcgi_pass unix:/var/run/php/php7.0-fpm.sock;
        fastcgi_index index.php;
        fastcgi_param SCRIPT_FILENAME $document_root$fastcgi_script_name;
        include fastcgi_params;
    }

使用Apache2

安装Apache2

$ sudo apt-get install apache

将/etc/phpmyadmin/apache.conf文件内容复制到Apache配置文件里，service apache2 reload

错误

不安装php-fpm会出现502错误，root软连接不能访问。。。
