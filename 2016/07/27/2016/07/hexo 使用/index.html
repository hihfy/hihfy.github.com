发布

注意：需要安装github插件npm install hexo-deployer-git --save

标签

在文章里面加上tag标签

tags:
  - iOS
  - Hexo
  - HTML
  - JavaScript

网页图标favicon

在主题的_config.yml文件里改，favicon: /favicon.ico，将图片放在source下面。

404页面

腾讯公益404，在source文件夹里面创建一个404.html，文件，粘贴下面代码就可。

CNAME

使用自己的域名，在域名DNS管理添加CNAME记录，指向github.io界面，在source文件夹里创建CNAME文件，直接github上创建，每次hexo d后都会消失。文件输入你的域名。

notes.hihfy.com

多电脑编辑

尝试过使用onedrive来进行多电脑直之间的多电脑之间的同步，最后发现不好用。将文件上传到github或者其他版本控制工具，.gitignore文件里面的内容最好添加不删除，换电脑后，clone下来，执行nmp install会自动去安装，hexo g、hexo s查看效果。

缓存问题

在发现一些数字或者文章显示不对，执行hexo clean，然后在hexo g、hexo s看看效果
